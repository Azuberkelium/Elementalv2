    Rules

    Objective: Form molecules by tapping and connecting the correct elements on the grid.

    How to Play:

    • A list of molecules to find will appear.
    • Tap on the elements in the grid to select them.
    • Connect elements by drawing a line. You can only connect adjacent elements (horizontally, vertically, or diagonally).
    • If the selected elements form one of the molecules, you earn points!
    • The tiles you used will be replaced with new random elements, so you can find the same molecules again.
    • Each successful combination earns you points based on the number of elements, plus a speed bonus.
    • Keep going until the timer runs out.
